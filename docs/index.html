Burrow - Mobile App

Burrow — Buurtcommunity App

Burrow is een buurtcommunity-app die bewoners van woonwijken en appartementsgebouwen digitaal met elkaar verbindt. Het platform is gebouwd rond drie pijlers: sociale cohesie, duurzaamheid en efficiëntie. Veel buren kennen elkaar nauwelijks, communiceren via versnipperde WhatsApp-groepen en verspillen spullen die anderen goed kunnen gebruiken. Burrow lost dit op met één gecentraliseerd platform.

Technisch bestaat de applicatie uit een Blazor Hybrid frontend gebouwd op .NET MAUI, gekoppeld aan een .NET Web API via een REST API. De gedeelde business logic en entiteiten leven in een apart Core-project. Realtime communicatie verloopt via Firestore, authenticatie via Firebase Authentication gecombineerd met JWT-tokens.

De kernfunctionaliteiten omvatten: communities met gelaagd lidmaatschap en moderatierollen, een buurtfeed met posts en likes, privéchat en community-chatrooms, een evenementenmodule met datumprikker en benodigdhedenlijst, een lokale marketplace voor verkoop en weggeven, een interactieve kaart met privacybeheer, en push-notificaties via Firebase Cloud Messaging.
